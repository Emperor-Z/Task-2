================================================================================
                         DEBUGGING TIPS
================================================================================

If tests fail during Phase 1:

1. Check CSV parsing:
   - Is CSV file readable?
   - Are column names correct? (Member_number, Date, itemDescription)
   - Are items being extracted correctly?

2. Check grouping:
   - Are items grouped by BOTH Member_number AND Date?
   - Is the grouping key (Member_number, Date) correct?

3. Check deduplication:
   - Use set() to automatically deduplicate
   - Convert back to list for sorting

4. Check sorting:
   - Use sorted() or list.sort()
   - Verify alphabetical order

5. Debug with print statements:
   - Print what you're reading from CSV
   - Print basket before and after sorting
   - Print final list of baskets
